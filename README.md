# ARK STUDIO · Mobile Design Studio

## Features
- ✅ 50+ container shapes with independent corner stretching
- ✅ Fill shapes with images + crop + opacity gradients
- ✅ Typography: 360° rotate, mirror, 3D gradients, bullet styles
- ✅ Backdrop shades with opacity dial
- ✅ Borders & Lines with full customization
- ✅ Undo/Redo with history cache
- ✅ Save/Load to localStorage
- ✅ Upload any file (JSON, DOCX, PDF, images)
- ✅ Complete offline support (PWA)
- ✅ GitHub Actions CI/CD
